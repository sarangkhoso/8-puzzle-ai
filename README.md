# 8-Puzzle AI Lab

A Streamlit app to solve the 8-puzzle using AI algorithms (BFS, DFS, UCS, Greedy, A*, IDA*).  
Styled tiles with animation and solution comparison.

## Deployment

1. Push this folder to GitHub.
2. Go to [Streamlit Cloud](https://share.streamlit.io/), log in with GitHub.
3. Click **New app**, select this repo and `puzzlefinal.py`.
4. Click **Deploy** and share the URL!
